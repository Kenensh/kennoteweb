React

React 是一個用於構建用戶界面的 JavaScript 庫，由 Facebook 開發並維護。 React is a JavaScript library for building user interfaces, developed and maintained by Facebook.

React 簡介 Introduction to React

React 是一個用於構建用戶界面的 JavaScript 庫，特別適合開發單頁應用程序。它允許開發者創建可重用的 UI 組件，並通過變更數據高效地更新和渲染適當的組件。

React is a JavaScript library for building user interfaces, particularly suited for single-page applications. It allows developers to create reusable UI components and efficiently update and render the appropriate components when data changes.

核心概念

Core Concepts

  • 組件 (Components) - React 應用由組件構建而成
  • Components - React applications are built from components
  • JSX - JavaScript 的語法擴展，允許在 JavaScript 中編寫類似 HTML 的代碼
  • JSX - JavaScript syntax extension that allows writing HTML-like code in JavaScript
  • Props - 用於將數據從父組件傳遞到子組件
  • Props - Used to pass data from parent to child components
  • 狀態 (State) - 組件的可變數據
  • State - Mutable data within components
  • 生命週期 (Lifecycle) - 組件的創建到銷毀的過程
  • Lifecycle - The process from creation to destruction of components

簡單的 React 組件示例 Simple React Component Example 

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

export default Counter;

上面的代碼展示了一個簡單的 React 函數組件，使用 useState Hook 來管理計數狀態。

The code above demonstrates a simple React functional component that uses the useState Hook to manage a count state.
