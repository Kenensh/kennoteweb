jQuery

jQuery 是一個快速、小型且功能豐富的 JavaScript 函式庫，讓 HTML 文檔操作、事件處理、動畫和 Ajax 變得更加簡單。 jQuery is a fast, small, and feature-rich JavaScript library that makes HTML document manipulation, event handling, animation, and Ajax much simpler.

jQuery 簡介 Introduction to jQuery

jQuery 是一個輕量級的 JavaScript 函式庫，由 John Resig 於 2006 年創建。它的核心理念是「寫得更少，做得更多」（Write Less, Do More），通過提供簡潔的 API 來簡化 JavaScript 編程。雖然現代 JavaScript 和框架（如 React、Vue）的興起使 jQuery 的使用有所減少，但它仍然是許多專案的重要組成部分。

jQuery is a lightweight JavaScript library created by John Resig in 2006. Its core philosophy is "Write Less, Do More," simplifying JavaScript programming by providing concise APIs. While the rise of modern JavaScript and frameworks (like React, Vue) has reduced jQuery usage, it remains an important component of many projects.

jQuery 基本語法

jQuery Basic Syntax

// 引入 jQuery (CDN)
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

// 文檔就緒事件
$(document).ready(function() {
    // 或簡寫為
    $(function() {
        console.log('jQuery is ready!');
    });
});

// 基本選擇器
$('#myId')          // ID 選擇器
$('.myClass')       // class 選擇器  
$('div')            // 標籤選擇器
$('div.myClass')    // 組合選擇器

// DOM 操作
$('#myDiv').text('Hello jQuery!');      // 設置文字
$('#myDiv').html('<b>Bold text</b>');    // 設置 HTML
$('#myDiv').addClass('highlight');      // 添加 class
$('#myDiv').css('color', 'red');        // 設置樣式

// 事件處理
$('#myButton').click(function() {
    alert('Button clicked!');
});

// 動畫效果
$('#myDiv').fadeIn();           // 淡入
$('#myDiv').slideDown();        // 滑動顯示
$('#myDiv').animate({           // 自定義動畫
    left: '100px',
    opacity: 0.5
}, 1000);

// AJAX 請求
$.ajax({
    url: '/api/data',
    method: 'GET',
    success: function(data) {
        console.log('Data received:', data);
    },
    error: function(xhr, status, error) {
        console.error('Error:', error);
    }
});

這是一個 jQuery 的基本語法示例，展示了選擇器、DOM 操作、事件處理和 AJAX 等核心功能。jQuery 的語法簡潔直觀，非常適合快速開發。

This is a basic jQuery syntax example showcasing core features like selectors, DOM manipulation, event handling, and AJAX. jQuery's syntax is concise and intuitive, perfect for rapid development.
