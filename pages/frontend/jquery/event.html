事件處理介紹

Event Handling Introduction

事件處理是 jQuery 的核心功能之一，它讓我們能夠響應用戶的互動行為，如點擊、鍵盤輸入、滑鼠移動等。jQuery 提供了統一且強大的事件處理系統，簡化了跨瀏覽器的事件處理。

Event handling is one of jQuery's core features, allowing us to respond to user interactions such as clicks, keyboard input, mouse movements, etc. jQuery provides a unified and powerful event handling system that simplifies cross-browser event handling.

jQuery 的事件系統建立在 JavaScript 原生事件之上，但提供了更簡潔的 API 和更好的跨瀏覽器兼容性。 jQuery's event system is built on top of native JavaScript events but provides a cleaner API and better cross-browser compatibility.

事件處理的基本概念

Basic Event Handling Concepts

// 事件處理的基本語法
$(selector).eventMethod(handler);

// 或使用通用的 on() 方法
$(selector).on(eventType, handler);

// 範例
$('#button').click(function() {
    alert('按鈕被點擊了！');
});

// 等同於
$('#button').on('click', function() {
    alert('按鈕被點擊了！');
});

事件對象的方法

Event Object Methods

$('#link').click(function(event) {
    // 阻止默認行為
    event.preventDefault();
    console.log('連結的默認跳轉被阻止');
    
    // 停止事件冒泡
    event.stopPropagation();
    console.log('事件冒泡被停止');
    
    // 同時阻止默認行為和事件冒泡
    event.stopImmediatePropagation();
    
    // 檢查是否阻止了默認行為
    if (event.isDefaultPrevented()) {
        console.log('默認行為已被阻止');
    }
    
    // 檢查是否停止了事件冒泡
    if (event.isPropagationStopped()) {
        console.log('事件冒泡已被停止');
    }
});

// 事件冒泡示例
$('#parent').click(function() {
    console.log('父元素被點擊');
});

$('#child').click(function(event) {
    console.log('子元素被點擊');
    
    // 如果不停止冒泡，父元素的點擊事件也會被觸發
    event.stopPropagation();
});

事件委託

Event Delegation

事件委託是一種強大的事件處理技術，特別適用於動態添加的元素：

Event delegation is a powerful event handling technique, especially useful for dynamically added elements:

基本事件委託

Basic Event Delegation

// 傳統方式 - 對現有元素綁定事件
$('.button').click(function() {
    console.log('按鈕被點擊');
});
// 問題：動態添加的按鈕不會有此事件

// 事件委託 - 對父容器綁定事件
$('#container').on('click', '.button', function() {
    console.log('按鈕被點擊（委託方式）');
});
// 優點：動態添加的按鈕也會有此事件

// 動態添加元素的範例
$('#addButton').click(function() {
    var newButton = $('<button class="button">新按鈕</button>');
    $('#container').append(newButton);
    // 新按鈕自動具有委託的點擊事件
});

複雜的事件委託

Complex Event Delegation

// 委託多個事件類型
$('#container').on('click mouseenter mouseleave', '.item', function(event) {
    switch(event.type) {
        case 'click':
            $(this).toggleClass('active');
            break;
        case 'mouseenter':
            $(this).addClass('hover');
            break;
        case 'mouseleave':
            $(this).removeClass('hover');
            break;
    }
});

// 委託不同的選擇器
$('#container').on({
    'click': function() {
        console.log('按鈕被點擊');
    },
    'mouseenter': function() {
        $(this).addClass('hover');
    },
    'mouseleave': function() {
        $(this).removeClass('hover');
    }
}, '.button');

// 委託表單事件
$('#form-container').on('submit', 'form', function(event) {
    event.preventDefault();
    console.log('表單提交：', $(this).attr('id'));
});

$('#form-container').on('change', 'input, select', function() {
    console.log('表單元素值改變：', $(this).attr('name'), $(this).val());
});

事件委託的優勢

Advantages of Event Delegation

// 1. 性能優勢 - 減少事件監聽器數量
// ❌ 為每個元素綁定事件（性能較差）
$('.item').each(function() {
    $(this).click(function() {
        // 處理點擊
    });
});

// ✅ 使用事件委託（性能更好）
$('#container').on('click', '.item', function() {
    // 處理點擊
});

// 2. 動態元素支持
// 動態添加的元素自動具有事件處理能力
function addItem(text) {
    var newItem = $('<div class="item">' + text + '</div>');
    $('#container').append(newItem);
    // 不需要重新綁定事件
}

// 3. 記憶體效率
// 只有一個事件監聽器，而不是 N 個
var itemCount = 1000;
for (var i = 0; i < itemCount; i++) {
    $('#container').append('<div class="item">Item ' + i + '</div>');
}
// 1000 個元素，但只有 1 個事件監聽器

// 4. 簡化事件管理
// 統一的事件處理邏輯
$('#container').on('click', '.delete-btn', function() {
    $(this).closest('.item').remove();
});

$('#container').on('click', '.edit-btn', function() {
    var item = $(this).closest('.item');
    item.find('.content').attr('contenteditable', true);
});

事件委託最佳實踐

Event Delegation Best Practices

// 1. 選擇合適的委託容器
// ✅ 選擇最近的靜態父元素
$('#user-list').on('click', '.user-item', handler);

// ❌ 不要委託到太遠的祖先元素
$(document).on('click', '.user-item', handler); // 過於廣泛

// 2. 使用具體的選擇器
// ✅ 具體的選擇器
$('#menu').on('click', '.menu-item.active', handler);

// ❌ 過於寬泛的選擇器
$('#menu').on('click', 'div', handler); // 可能影響其他 div

// 3. 組合相關事件
$('#data-table').on({
    'click': function() {
        // 處理行點擊
        $(this).addClass('selected').siblings().removeClass('selected');
    },
    'dblclick': function() {
        // 處理行雙擊
        editRow($(this));
    },
    'contextmenu': function(event) {
        // 處理右擊
        event.preventDefault();
        showContextMenu(event.pageX, event.pageY);
    }
}, '.table-row');

// 4. 避免過度委託
// ✅ 對頻繁變化的元素使用委託
$('#dynamic-content').on('click', '.dynamic-button', handler);

// ❌ 對靜態元素不必要地使用委託
$('#static-nav').on('click', '.nav-link', handler); // 直接綁定更好

自定義事件實際應用

Practical Custom Event Applications

// 購物車系統的自定義事件
var ShoppingCart = {
    items: [],
    
    init: function() {
        this.bindEvents();
    },
    
    bindEvents: function() {
        // 監聽購物車事件
        $(document).on('cart:itemAdded', this.onItemAdded.bind(this));
        $(document).on('cart:itemRemoved', this.onItemRemoved.bind(this));
        $(document).on('cart:cleared', this.onCartCleared.bind(this));
    },
    
    addItem: function(item) {
        this.items.push(item);
        $(document).trigger('cart:itemAdded', [item]);
    },
    
    removeItem: function(itemId) {
        this.items = this.items.filter(item => item.id !== itemId);
        $(document).trigger('cart:itemRemoved', [itemId]);
    },
    
    clear: function() {
        this.items = [];
        $(document).trigger('cart:cleared');
    },
    
    onItemAdded: function(event, item) {
        this.updateCartDisplay();
        this.showNotification('商品已添加到購物車');
    },
    
    onItemRemoved: function(event, itemId) {
        this.updateCartDisplay();
        this.showNotification('商品已從購物車移除');
    },
    
    onCartCleared: function() {
        this.updateCartDisplay();
        this.showNotification('購物車已清空');
    }
};

// 表單驗證系統的自定義事件
var FormValidator = {
    init: function() {
        this.bindEvents();
    },
    
    bindEvents: function() {
        $(document).on('form:validate', this.validateForm.bind(this));
        $(document).on('field:validate', this.validateField.bind(this));
        $(document).on('validation:success', this.onValidationSuccess.bind(this));
        $(document).on('validation:error', this.onValidationError.bind(this));
    },
    
    validateForm: function(event, form) {
        var isValid = true;
        var errors = [];
        
        $(form).find('[required]').each(function() {
            var field = $(this);
            var value = field.val().trim();
            
            if (!value) {
                isValid = false;
                errors.push({
                    field: field.attr('name'),
                    message: '此欄位為必填'
                });
            }
        });
        
        if (isValid) {
            $(document).trigger('validation:success', [form]);
        } else {
            $(document).trigger('validation:error', [form, errors]);
        }
    },
    
    onValidationSuccess: function(event, form) {
        $(form).find('.error').removeClass('error');
        $(form).find('.error-message').remove();
    },
    
    onValidationError: function(event, form, errors) {
        errors.forEach(function(error) {
            var field = $(form).find('[name="' + error.field + '"]');
            field.addClass('error');
            field.after('<span class="error-message">' + error.message + '</span>');
        });
    }
};

這些高級技巧可以幫助您更有效地處理複雜的事件場景：

事件節流和防抖

Event Throttling and Debouncing

// 防抖函數 - 延遲執行，頻繁觸發時重新計時
function debounce(func, wait) {
    var timeout;
    return function() {
        var context = this;
        var args = arguments;
        clearTimeout(timeout);
        timeout = setTimeout(function() {
            func.apply(context, args);
        }, wait);
    };
}

// 節流函數 - 限制執行頻率
function throttle(func, wait) {
    var timeout;
    return function() {
        var context = this;
        var args = arguments;
        if (!timeout) {
            timeout = setTimeout(function() {
                timeout = null;
                func.apply(context, args);
            }, wait);
        }
    };
}

// 使用防抖處理搜索輸入
$('#search-input').on('keyup', debounce(function() {
    var query = $(this).val();
    if (query.length > 2) {
        performSearch(query);
    }
}, 300));

// 使用節流處理滾動事件
$(window).on('scroll', throttle(function() {
    var scrollTop = $(this).scrollTop();
    updateScrollPosition(scrollTop);
}, 100));

// 使用節流處理窗口大小改變
$(window).on('resize', throttle(function() {
    var width = $(this).width();
    var height = $(this).height();
    updateLayout(width, height);

事件隊列和批次處理

Event Queue and Batch Processing

// 事件隊列管理
var EventQueue = {
    queue: [],
    processing: false,
    
    add: function(eventData) {
        this.queue.push(eventData);
        this.process();
    },
    
    process: function() {
        if (this.processing || this.queue.length === 0) {
            return;
        }
        
        this.processing = true;
        
        // 批次處理事件
        var batch = this.queue.splice(0, 10); // 每次處理 10 個事件
        
        batch.forEach(function(eventData) {
            this.handleEvent(eventData);
        }.bind(this));
        
        this.processing = false;
        
        // 繼續處理剩餘事件
        if (this.queue.length > 0) {
            setTimeout(this.process.bind(this), 0);
        }
    },
    
    handleEvent: function(eventData) {
        // 處理單個事件
        console.log('處理事件：', eventData);
    }
};

// 批次處理 DOM 更新
var DOMUpdater = {
    updates: [],
    
    scheduleUpdate: function(element, property, value) {
        this.updates.push({
            element: element,
            property: property,
            value: value
        });
        
        // 使用 requestAnimationFrame 批次更新
        if (!this.scheduled) {
            this.scheduled = true;
            requestAnimationFrame(this.applyUpdates.bind(this));
        }
    },
    
    applyUpdates: function() {
        this.updates.forEach(function(update) {
            $(update.element).css(update.property, update.value);
        });
        
        this.updates = [];
        this.scheduled = false;
    }
};

// 事件記錄器
var EventLogger = {
    logs: [],
    enabled: true,
    
    log: function(eventType, element, data) {
        if (!this.enabled) return;
        
        var logEntry = {
            timestamp: new Date().toISOString(),
            type: eventType,
            element: element.tagName + (element.id ? '#' + element.id : ''),
            data: data || {}
        };
        
        this.logs.push(logEntry);
        console.log('Event:', logEntry);
        
        // 限制日誌數量
        if (this.logs.length > 1000) {
            this.logs = this.logs.slice(-500);
        }
    },
    
    getRecentLogs: function(count) {
        return this.logs.slice(-count || -10);
    },
    
    clearLogs: function() {
        this.logs = [];
    }
};

// 包裝事件處理器以添加日誌

    };
}

// ✅ 使用模塊模式封裝事件處理
var EventManager = (function() {
    var handlers = {};
    
    function bindEvent(selector, eventType, handler) {
        var key = selector + ':' + eventType;
        handlers[key] = handler;
        $(document).on(eventType, selector, handler);
    }
    
    function unbindEvent(selector, eventType) {
        var key = selector + ':' + eventType;
        if (handlers[key]) {
            $(document).off(eventType, selector, handlers[key]);
            delete handlers[key];
        }
    }
    
    return {
        bind: bindEvent,
        unbind: unbindEvent,
        handlers: handlers
    };
})();
