JavaScript

JavaScript 是現代網頁開發的核心語言，也是全端開發者的必備技能。 JavaScript is the core language of modern web development and an essential skill for full-stack developers.

JavaScript 簡介 Introduction to JavaScript

JavaScript 是一種高級、解釋型的編程語言，最初設計用於為網頁添加交互性。如今，JavaScript 已經發展成為一種多功能的編程語言，不僅可以用於前端開發，還可以用於後端開發（Node.js）、移動應用開發、桌面應用開發等領域。

JavaScript is a high-level, interpreted programming language originally designed to add interactivity to web pages. Today, JavaScript has evolved into a versatile programming language that can be used not only for frontend development but also for backend development (Node.js), mobile app development, desktop app development, and more.

JavaScript 的特點

Features of JavaScript

  • 動態類型 - 變數類型在運行時確定
  • Dynamic Typing - Variable types are determined at runtime
  • 原型繼承 - 基於原型鏈的物件導向編程
  • Prototype Inheritance - Object-oriented programming based on prototype chain
  • 函數式編程 - 支持高階函數和閉包
  • Functional Programming - Supports higher-order functions and closures
  • 事件驅動 - 異步編程和事件處理
  • Event-Driven - Asynchronous programming and event handling
  • 跨平台 - 可在瀏覽器、服務器和移動設備運行
  • Cross-Platform - Runs on browsers, servers, and mobile devices

JavaScript 基本語法

JavaScript Basic Syntax

// 變數宣告
let name = 'JavaScript';
const version = 'ES2024';
var isAwesome = true;

// 函數定義
function greet(name) {
    return `Hello, ${name}!`;
}

// 箭頭函數
const add = (a, b) => a + b;

// 物件與陣列
const person = {
    name: 'Alice',
    age: 30,
    skills: ['JavaScript', 'React', 'Node.js']
};

// 解構賦值
const { name: personName, skills } = person;
const [firstSkill, ...otherSkills] = skills;

// 異步操作
async function fetchData() {
    try {
        const response = await fetch('/api/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

這是一個 JavaScript 的基本語法示例，涵蓋了變數宣告、函數定義、物件操作和異步編程等核心概念。
