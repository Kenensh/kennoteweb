選擇器介紹

Selector Introduction

jQuery 選擇器是 jQuery 的核心功能之一，它基於 CSS 選擇器語法，並提供了額外的功能和擴展。選擇器讓我們能夠精確地定位和操作 HTML 元素，是所有 jQuery 操作的基礎。

jQuery selectors are one of jQuery's core features, based on CSS selector syntax with additional functionality and extensions. Selectors allow us to precisely locate and manipulate HTML elements, forming the foundation of all jQuery operations.

掌握選擇器是學習 jQuery 的關鍵，不同的選擇器在性能上有很大差異，正確選擇能大幅提升代碼效率。 Mastering selectors is key to learning jQuery. Different selectors vary greatly in performance, and choosing correctly can significantly improve code efficiency.

選擇器的基本語法

Basic Selector Syntax

// jQuery 選擇器的基本結構
$(selector)

// 範例
$('#myId')        // 選擇 ID 為 myId 的元素
$('.myClass')     // 選擇 class 為 myClass 的元素
$('div')          // 選擇所有 div 元素
$('*')            // 選擇所有元素

基本選擇器

Basic Selectors

基本選擇器是最常用的選擇器類型，對應 CSS 的基本選擇器：

Basic selectors are the most commonly used selector types, corresponding to CSS basic selectors:

元素選擇器

Element Selectors

// 標籤名選擇器
$('p')            // 選擇所有 p 標籤
$('div')          // 選擇所有 div 標籤
$('span')         // 選擇所有 span 標籤
$('h1')           // 選擇所有 h1 標籤

// 通用選擇器
$('*')            // 選擇頁面中所有元素（慎用，性能較差）

// 多選擇器
$('h1, h2, h3')   // 選擇所有 h1、h2、h3 標籤
$('p, div, span') // 選擇所有 p、div、span 標籤

ID 選擇器

ID Selectors

// ID 選擇器（最快的選擇器）
$('#header')      // 選擇 ID 為 header 的元素
$('#main')        // 選擇 ID 為 main 的元素
$('#footer')      // 選擇 ID 為 footer 的元素

// ID 應該是唯一的，每個頁面只能有一個相同的 ID
// 如果有多個相同 ID，只會選擇第一個

Class 選擇器

Class Selectors

// Class 選擇器
$('.highlight')   // 選擇所有 class 為 highlight 的元素
$('.btn')         // 選擇所有 class 為 btn 的元素
$('.active')      // 選擇所有 class 為 active 的元素

// 多個 class 的元素
$('.btn.primary') // 選擇同時有 btn 和 primary class 的元素
$('.menu.open')   // 選擇同時有 menu 和 open class 的元素

組合選擇器

Combined Selectors

// 元素 + Class
$('div.container')    // 選擇 class 為 container 的 div 元素
$('p.intro')          // 選擇 class 為 intro 的 p 元素

// 元素 + ID
$('div#main')         // 選擇 ID 為 main 的 div 元素
$('section#about')    // 選擇 ID 為 about 的 section 元素

// 複雜組合
$('div.container.main')   // 選擇同時有 container 和 main class 的 div
$('input.form-control.required') // 選擇有多個 class 的 input

層級選擇器

Hierarchy Selectors

層級選擇器用於選擇元素之間的層級關係：

Hierarchy selectors are used to select elements based on their hierarchical relationships:

後代選擇器

Descendant Selectors

// 後代選擇器（空格分隔）
$('div p')        // 選擇 div 內的所有 p 元素（包括孫輩）
$('ul li')        // 選擇 ul 內的所有 li 元素
$('.container p') // 選擇 class 為 container 的元素內的所有 p

// 多層後代
$('div p span')   // 選擇 div 內 p 元素中的所有 span
$('#header .nav a') // 選擇 ID 為 header 內 class 為 nav 的元素中的所有 a

子元素選擇器

Child Selectors

// 子元素選擇器（> 分隔）
$('div > p')      // 選擇 div 的直接子元素中的 p（不包括孫輩）
$('ul > li')      // 選擇 ul 的直接子元素中的 li
$('.menu > a')    // 選擇 class 為 menu 的元素的直接子元素中的 a

// 與後代選擇器的區別
$('div p')        // 選擇 div 內所有 p（包括孫輩）
$('div > p')      // 只選擇 div 的直接子元素 p

相鄰兄弟選擇器

Adjacent Sibling Selectors

// 相鄰兄弟選擇器（+ 分隔）
$('h2 + p')      // 選擇緊接在 h2 後面的第一個 p 元素
$('label + input') // 選擇緊接在 label 後面的第一個 input
$('.title + .content') // 選擇緊接在 title 後面的 content

// 只選擇直接相鄰的下一個元素

一般兄弟選擇器

General Sibling Selectors

// 一般兄弟選擇器（~ 分隔）
$('h2 ~ p')      // 選擇 h2 後面所有同級的 p 元素
$('label ~ input') // 選擇 label 後面所有同級的 input
$('.title ~ .item') // 選擇 title 後面所有同級的 item

// 選擇所有符合條件的兄弟元素，不僅僅是相鄰的

層級選擇器範例

Hierarchy Selector Examples

// HTML 結構範例
<div class="container">
  <h2>標題</h2>
  <p>第一段</p>
  <div class="content">
    <p>內容段落</p>
    <span>內容</span>
  </div>
  <p>第二段</p>
</div>

// 不同選擇器的結果
$('.container p')     // 選擇 3 個 p 元素（包括 .content 內的）
$('.container > p')   // 選擇 2 個 p 元素（不包括 .content 內的）
$('h2 + p')          // 選擇 1 個 p 元素（第一段）
$('h2 ~ p')          // 選擇 2 個 p 元素（第一段和第二段）

過濾選擇器

Filter Selectors

過濾選擇器用於進一步細化選擇結果：

Filter selectors are used to further refine selection results:

位置過濾器

Position Filters

// 基本位置過濾器
$('li:first')         // 選擇第一個 li 元素
$('li:last')          // 選擇最後一個 li 元素
$('li:eq(2)')         // 選擇第三個 li 元素（索引從 0 開始）
$('li:gt(1)')         // 選擇索引大於 1 的 li 元素
$('li:lt(3)')         // 選擇索引小於 3 的 li 元素

// nth-child 過濾器
$('li:nth-child(2)')     // 選擇每個父元素的第二個 li 子元素
$('li:nth-child(odd)')   // 選擇奇數位置的 li 元素
$('li:nth-child(even)')  // 選擇偶數位置的 li 元素
$('li:nth-child(3n)')    // 選擇每第三個 li 元素

// first-child 和 last-child
$('p:first-child')    // 選擇作為第一個子元素的 p
$('p:last-child')     // 選擇作為最後一個子元素的 p
$('p:only-child')     // 選擇作為唯一子元素的 p

內容過濾器

Content Filters

// 內容相關過濾器
$(':contains("jQuery")')    // 選擇包含文字 "jQuery" 的元素
$(':empty')                 // 選擇空元素（沒有子元素和文字）
$(':parent')                // 選擇有子元素或文字的元素

// 範例
$('p:contains("重要")')     // 選擇包含 "重要" 文字的 p 元素
$('div:empty')              // 選擇空的 div 元素
$('td:parent')              // 選擇有內容的 td 元素

可見性過濾器

Visibility Filters

// 可見性過濾器
$(':visible')           // 選擇可見的元素
$(':hidden')            // 選擇隱藏的元素

// 範例
$('div:visible')        // 選擇可見的 div 元素
$('p:hidden')           // 選擇隱藏的 p 元素

// 注意：hidden 包括 display:none、visibility:hidden、height/width 為 0 等情況

自定義過濾器

Custom Filters

// jQuery 特有的過濾器
$(':header')            // 選擇所有標題元素（h1-h6）
$(':animated')          // 選擇正在執行動畫的元素
$(':focus')             // 選擇當前獲得焦點的元素

// 範例
$(':header:contains("jQuery")')  // 選擇包含 "jQuery" 的標題元素
$('div:animated')                // 選擇正在動畫的 div 元素

屬性選擇器

Attribute Selectors

屬性選擇器根據元素的屬性來選擇元素：

Attribute selectors select elements based on their attributes:

基本屬性選擇器

Basic Attribute Selectors

// 有屬性的元素
$('[title]')              // 選擇有 title 屬性的元素
$('[href]')               // 選擇有 href 屬性的元素
$('[data-id]')            // 選擇有 data-id 屬性的元素

// 屬性值完全匹配
$('[title="hello"]')      // 選擇 title 屬性值為 "hello" 的元素
$('[type="text"]')        // 選擇 type 屬性值為 "text" 的元素
$('[class="btn"]')        // 選擇 class 屬性值為 "btn" 的元素

// 屬性值不等於
$('[title!="hello"]')     // 選擇 title 屬性值不為 "hello" 的元素
$('[type!="hidden"]')     // 選擇 type 屬性值不為 "hidden" 的元素

模糊匹配屬性選擇器

Pattern Matching Attribute Selectors

// 屬性值包含指定字串
$('[title*="hello"]')     // 選擇 title 屬性值包含 "hello" 的元素
$('[class*="btn"]')       // 選擇 class 屬性值包含 "btn" 的元素

// 屬性值以指定字串開始
$('[title^="hello"]')     // 選擇 title 屬性值以 "hello" 開始的元素
$('[href^="http"]')       // 選擇 href 屬性值以 "http" 開始的元素
$('[href^="mailto"]')     // 選擇 href 屬性值以 "mailto" 開始的元素

// 屬性值以指定字串結束
$('[title$="world"]')     // 選擇 title 屬性值以 "world" 結束的元素
$('[href$=".pdf"]')       // 選擇 href 屬性值以 ".pdf" 結束的元素
$('[src$=".jpg"]')        // 選擇 src 屬性值以 ".jpg" 結束的元素

// 屬性值包含指定單詞（以空格分隔）
$('[title~="important"]') // 選擇 title 屬性值包含單詞 "important" 的元素
$('[class~="active"]')    // 選擇 class 屬性值包含單詞 "active" 的元素

// 屬性值等於指定值或以指定值開頭後跟連字符
$('[lang|="en"]')         // 選擇 lang 屬性值為 "en" 或 "en-*" 的元素

多屬性選擇器

Multiple Attribute Selectors

// 同時匹配多個屬性
$('[type="text"][name="username"]')  // 選擇 type 為 text 且 name 為 username 的元素
$('[class="btn"][disabled]')         // 選擇 class 為 btn 且有 disabled 屬性的元素
$('[href^="http"][target="_blank"]') // 選擇外部連結且在新視窗開啟的元素

// 結合元素選擇器
$('input[type="text"]')              // 選擇 type 為 text 的 input 元素
$('a[href^="http"]')                 // 選擇外部連結的 a 元素
$('img[alt*="photo"]')               // 選擇 alt 屬性包含 "photo" 的 img 元素

實用的屬性選擇器範例

Practical Attribute Selector Examples

// 選擇所有外部連結
$('a[href^="http"]')

// 選擇所有 PDF 連結
$('a[href$=".pdf"]')

// 選擇所有電子郵件連結
$('a[href^="mailto"]')

// 選擇所有必填欄位
$('input[required]')

// 選擇所有禁用的按鈕
$('button[disabled]')

// 選擇所有有 placeholder 的輸入框
$('input[placeholder]')

// 選擇所有 data 屬性
$('[data-toggle]')           // 有 data-toggle 屬性的元素
$('[data-role="button"]')    // data-role 為 button 的元素

// 選擇特定 ID 模式的元素
$('[id^="user-"]')           // ID 以 "user-" 開頭的元素
$('[id$="-form"]')           // ID 以 "-form" 結尾的元素

表單選擇器實際應用

Practical Form Selector Applications

// 表單驗證
function validateForm() {
    // 檢查所有必填欄位
    var emptyFields = $('input[required]').filter(function() {
        return $(this).val() === '';
    });
    
    if (emptyFields.length > 0) {
        emptyFields.addClass('error');
        return false;
    }
    
    // 檢查電子郵件格式
    var emailFields = $('input[type="email"]');
    var invalidEmails = emailFields.filter(function() {
        var email = $(this).val();
        return email && !isValidEmail(email);
    });
    
    if (invalidEmails.length > 0) {
        invalidEmails.addClass('error');
        return false;
    }
    
    return true;
}

// 重設表單
function resetForm() {
    $('input[type="text"], input[type="email"], textarea').val('');
    $('input[type="checkbox"], input[type="radio"]').prop('checked', false);
    $('select').prop('selectedIndex', 0);
    $('.error').removeClass('error');
}

// 禁用/啟用表單
function toggleForm(disabled) {
    $('input, textarea, select, button').prop('disabled', disabled);
}

// 收集表單數據
function getFormData() {
    var data = {};
    
    // 文字輸入框
    $('input[type="text"], input[type="email"], textarea').each(function() {
        data[$(this).attr('name')] = $(this).val();
    });
    
    // 複選框
    $('input[type="checkbox"]:checked').each(function() {
        data[$(this).attr('name')] = $(this).val();
    });
    
    // 單選按鈕
    $('input[type="radio"]:checked').each(function() {
        data[$(this).attr('name')] = $(this).val();
    });
    
    // 下拉選單
    $('select').each(function() {
        data[$(this).attr('name')] = $(this).val();
    });
    
    return data;
}

最佳實踐

Best Practices

// ✅ 好的做法

// 1. 使用 ID 選擇器當可能時
$('#header')  // 快

// 2. 限制搜索範圍
$('#container').find('.item')  // 快
$('#container .item')          // 較慢

// 3. 快取選擇器結果
var $items = $('.item');
$items.addClass('active');
$items.show();

// 4. 使用具體的選擇器
$('input[type="text"]')  // 好
$(':text')               // 較慢（jQuery 特有選擇器）

// 5. 避免過度具體的選擇器
$('.container .item')    // 好
$('body .container .nav .item')  // 過度具體

// ❌ 不好的做法

// 1. 重複查詢
$('.item').addClass('active');
$('.item').show();  // 重複查詢

// 2. 使用低效的選擇器
$('*')              // 選擇所有元素，非常慢
$(':visible')       // 需要檢查每個元素的可見性
$('[class*="btn"]') // 模糊匹配，較慢

// 3. 過度複雜的選擇器
$('div.container > ul li:nth-child(odd) a[href^="http"]')

function testSelectorPerformance() {

        $('#myId');

        $('.myClass');

    // 測試複雜選擇器
    start = Date.now();

    }

}
